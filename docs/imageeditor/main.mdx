## 템플릿을 선택한 경우
1. 모달 입력 (타입, 제목, 설명)
2. 추가 적인 프로모션 정보 입력
3. 사이즈 선택
4. 이미지 입력 & 생성 -> 이미지가 없을 수도 있음
5. AI 실행
6. 편집창

## 템플릿을 선택하지 않은 경우
1. 탬플릿 선택
2. 모달 입력 (타입, 제목, 설명)
3. 다음 세 가지 입력
    - template JSON
    - 유저가 채워넣은 layer 정보 -> 위치, 크기 속성만 AI가 변경
    - 내용 / 속성을 바꿔야 하는 레이어 정보 -> 모든걸 AI 가 알아서
