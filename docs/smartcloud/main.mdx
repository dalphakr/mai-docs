# 파일 업로드 방법

- **파일 업로드 / 폴더 업로드 버튼** <br/>
중앙 하다 업로드 버튼을 클릭하면 업로드 관련 모달창이 표시 됩니다.
- **드래그 앤 드롭** <br/>
업로드하고 싶은 파일을 드래그 앤 드롭으로 영역에 올리면 파일이 업로드 됩니다.


### 가운데 버튼
- collection 생성

### 오른쪽 버튼
- 배너랑 문서 생성


### 오른쪽 토글
- 화면 정보를 인지하고 있지 못함

### 오른쪽 위 검색
- 파일 내 검색? 그냥 전체 검색이랑 뭐가 다르지


### 그냥 클릭
- 오른쪽 위 버튼 누르면 FAQ 챗봇 가능
- 오른쪽 위에서 컬렉션 추가 가능
- 이름 변경
- 폴더 정보 / 파일 속성 및 정보
- 폴더 / 파일 삭제
- 권한 설정
- 고정 기능
- 컬렉션 생성 기능


### 더블 클릭
- 더블 클릭하면 들어가게 됨
- 그냥 view or Image editor 연결 가능
