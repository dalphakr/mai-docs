# AIMS News
<Card title="대홍기획, 마케팅 전용 올인원 AI 시스템 '에임스' 공개" href="https://www.yna.co.kr/view/AKR20240722048400003">
대홍기획은 마케팅 전용 올인원 인공지능(AI) 시스템 '에임스'(AIMS)를 공개하고 시연회를 열었다고 22일 밝혔다.
에임스는 AI, 빅데이터, 클라우드 기술을 적용해 하나의 플랫폼에서 기업의 광고 마케팅 전 과정을 망라한 다양한 기능을 실행한다.
