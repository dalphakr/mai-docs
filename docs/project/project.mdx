## Overview
프로젝트는 AIMS릴 이용하는 기본 공간입니다. 프로젝트를 생성 후 파일을 업로드하고 관련된 사람들을 초대해 함께 파일을 만들고 관리할 수 있습니ㅏㄷ.
MAI의 모든 AI는 프로젝트 내에서 실행됩니다. AI는 프로젝트 내의 파일을 참조해 최적의 결과를 제공하며 모든 AI의 실행기록은 프로젝트 내에 저장됩니다.

MAI 프로젝트는 실제 어떤 기업 혹은 서비스의 "마케팅 프로젝트"와 대응시켜 사용할 때 가장 편리하게 사용할 수 있습니다. 프로젝트에 참여하는 모든 사람들을 초대하고, 프로젝트와 관련된 파일을 모두 한 곳에 모아 관리해 보세요. 프로젝트의 모든 정보가 한 곳에 모였을 때 AI와 사람 모두 최고의 효율로 최고의 성과를 낼 수 있습니다.
